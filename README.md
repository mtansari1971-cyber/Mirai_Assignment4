# 🎨 AI Image Studio

An AI-powered Image Generation web application built using **Streamlit** and **Pollinations AI**. Users can generate creative images from text prompts, customize image dimensions, apply AI prompt enhancement, and download the generated images.

---

## Features

- Generate AI images from text prompts
- Adjustable Width and Height sliders
- Multiple Art Style options
- Magic Enhance feature for high-quality prompts
- Surprise Me button for random creative prompts
- Download generated images as `.png`
- Simple and interactive Streamlit interface

---

##  Technologies Used

- Python
- Streamlit
- Requests
- Pollinations AI API
- Random Module


## How to Use

1. Enter a text prompt.
2. Select your preferred Art Style.
3. Adjust Width and Height.
4. (Optional) Enable **Magic Enhance**.
5. Click **Generate Image**.
6. Or click **🎲 Surprise Me!** for a random prompt.
7. Download the generated image as a PNG file.


##  Assignment Features Implemented

###  Task 1 – Width & Height Sliders

The selected image dimensions are sent to the Pollinations AI API using URL parameters.

###  Task 2 – File Extension Fix

Downloaded images are automatically saved as:

```
<ArtStyle>_image.png
```

###  Task 3 – Magic Enhance

When enabled, the application automatically adds descriptive keywords to improve image quality.

Added keywords:

- masterpiece
- 8k resolution
- highly detailed
- trending on artstation
- unreal engine 5 render

###  Task 4 – Surprise Me

Generates an image using one of several randomly selected creative prompts.
